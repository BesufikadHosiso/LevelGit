# LevelGit

Version control your developer growth. A direct, no-fluff workflow for architecting roadmaps, locking into deep work, and committing every breakthrough to memory.

---

## The Workflow

- 🚀 **Commit to Today**: Manage your daily build goals and keep the streak alive.
- 🔒 **Deep Work Sessions**: Integrated focus timers to help you stay locked in.
- 🗺️ **Milestone Roadmaps**: Architect learning paths and track your progress through any stack.
- 📝 **Insight Logs**: Commit what you understand to your learning history.
- 📊 **Growth Stats**: Visualize your contribution heatmap and stack breakdown.
- 💾 **Local-First**: Total privacy with `localStorage` persistence.

---

## Stack

- **Core:** React 19 + Vite
- **Routing:** React Router
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **State:** React Context + `useReducer`

---

## Philosophy

- **Commit daily.** Momentum is better than motivation.
- **Log the struggle.** Documentation is your brain's secondary cache.
- **Stay focused.** One task, one timer, zero distractions.
- **Own your progress.** Local-first, privacy-by-default.
